# 潇潇烟雨 - Douluo Dalu H5 Private Deployment


## 项目说明

本项目仅包含游戏的前端静态资源（HTML / CSS / JavaScript / 图片），**不包含任何后端服务**。所有资源均引用自原始 CDN，部署后即可通过浏览器访问。
